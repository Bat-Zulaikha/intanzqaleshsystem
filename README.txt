---------------------------------------Installation Manual------------------------------------------------------------------------------------------------

1) Find XAMPP application online and install to computer.
2) Open XAMPP Control Panel on Windows and start the Apache and MySQL to initially startup the system.


---------------------------------------Opening System-----------------------------------------------------------------------------------------------------

1) Open the GD link -> https://drive.google.com/drive/folders/1FjR5ku5lx7NKdGQHclkcDSRpLrjXGH_E?usp=sharing
2) Find the 'System Folder' and download the 'intanzqaleshsystem' folder into XAMPP folder under the name 'htdocs'.
3) From XAMPP Control Panel, find MySQL row and click Admin to go to phpMyAdmin/localhost browser site.
4) In 'intanzqaleshsystem' folder under the name 'htdocs', find 'intanzQaleshSystemSql.sql' and import the sql code to the database in the phpMyAdmin/localhost site.
5) Paste the links of Intanz Qalesh System for Website and run.


---------------------------------------Links of Intanz Qalesh System for Website---------------------------------------------------------------------------

1) Customer Page -> http://localhost/intanzqaleshsystem/index.php
2) Admin Page -> http://localhost/intanzqaleshsystem/admin/adminindex.php

---------------------------------------If want to import some existing data into Localhost-----------------------------------------------------------------

1) Find 'Existing Data' folder in GD and download which file you want to import.
2) Open phpMyAdmin/localhost site and go to the respective table and find the Import Button in ribbon navigation and choose csv as the file type, upload the downloaded file and click GO.
3) Data in the file is now imported into database.
